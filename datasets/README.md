# Shared Datasets

Downloaded datasets for the entire `machine_learning` repository are stored
here so projects can reuse them without downloading duplicate copies.

Each project resolves this directory from its source file location rather than
from the current working directory. Dataset contents are ignored by Git.

Current dataset namespaces:

- `mnist/`
- `fashion_mnist/`
- `cifar10/`
- `cifar100/`
- `flowers102/`
- `ridership/`
- `tiny_shakespeare/`
